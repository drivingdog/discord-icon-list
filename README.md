# Discord Game Icons Gallery

A sleek Y2K-styled web gallery displaying Discord's detectable game icons. Browse through thousands of game icons with smooth infinite scrolling and easy one-click copying of game names to customize your Discord profile.

## Features

-  **Comprehensive Game Library** - Displays all Discord detectable games with icons
-  **One-Click Copy** - Click any game card to instantly copy its name to clipboard

## Usage

1. Install node (If its not installed) and run `node server.js`. 
2. Open your web browser and go to `localhost:3000`.

## Made with

- Pure HTML/CSS/JavaScript
- Tailwind CSS
- Discord Detectables endpoint `https://discord.com/api/applications/detectable`.
